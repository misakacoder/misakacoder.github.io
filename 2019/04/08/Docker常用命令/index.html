Docker常用命令

发表于 2019-04-08 | 更新于 2021-02-24 | 分类于 学习

1.Dockerfile

指定基础镜像：FROM <image>:<tag>

将文件添加到容器内：ADD <文件绝对路径 or 文件相对路径 or 网络文件路径> <镜像内部路径>

将文件添加到容器内：COPY <文件绝对路径 or 文件相对路径> <镜像内部路径>

设置环境变量：ENV <key1>=<value1> <key2>=<value2> ...

容器启动时执行命令：CMD ["executable","param1","param2"]

暴露容器运行时的监听端口给外部：EXPOSE <端口号>

2.docker命令

搜索镜像：docker search <镜像名>:<镜像版本号>

拉取镜像(不指定版本号则拉取latest)：docker pull <镜像名>:<镜像版本号>

创建镜像(在Dockerfile所在路径下执行)：docker build -t <镜像名>:<镜像版本号> .

查看所有镜像：docker images

创建容器：docker run --name <指定名称，可以不填> -d -p <外部端口>:<容器开放端口> -e <环境变量key>=<环境变量value> -e <环境变量key>=<环境变量value> <镜像名>:<镜像版本号>

查看运行中的容器：docker ps

查看所有容器：docker ps -a

进入容器内部：docker exec -i -t <容器名称 or 容器ID> /bin/bash

启动容器：docker start <容器名称 or 容器ID>

停止容器：docker stop <容器名称 or 容器ID>

重启容器：docker restart <容器名称 or 容器ID>

删除容器：docker rm <容器名称 or 容器ID>

删除镜像：docker rmi <容器名称 or 容器ID>

ps：要删除容器，必须先停止容器，要删除镜像，必须先删除容器

3.使用docker-compose构建镜像

安装docker-compose：
curl -L https://github.com/docker/compose/releases/download/1.24.0/docker-compose-`uname -s`-`uname -m` -o /usr/local/bin/docker-compose
chmod +x /usr/local/bin/docker-compose

docker-compose.yml示例：

构建镜像：docker-compose build

构建镜像并启动容器(在docker-compose.yml目录下执行)：docker-compose up -d

构建镜像并启动容器(指定docker-compose.yml)：docker-compose -f /home/docker/docker-compose.yml up -d

docker-compose查看容器： docker-compose -f /home/docker/docker-compose.yml ps

docker-compose启动容器：docker-compose -f /home/docker/docker-compose.yml start

docker-compose重启容器：docker-compose -f /home/docker/docker-compose.yml restart

docker-compose停止容器：docker-compose -f /home/docker/docker-compose.yml stop

docker-compose停止容器、删除容器：docker-compose -f /home/docker/docker-compose.yml down

docker-compose停止容器、删除容器、删除镜像：docker-compose -f /home/docker/docker-compose.yml down --rmi all

docker-compose停止容器、删除容器、删除镜像名为空的镜像：docker-compose -f /home/docker/docker-compose.yml down --rmi local

4.docker安装mysql

在https://hub.docker.com/_/mysql?tab=tags上查看mysql镜像有哪些版本

拉取mysql镜像：docker pull mysql:<镜像版本号>

创建mysql容器：docker run --name mysql -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root mysql:<镜像版本号> --character-set-server=utf8 --collation-server=utf8_bin --lower_case_table_names=1

ps：
设置root用户的密码为root：-e MYSQL_ROOT_PASSWORD=root
解决mysql中文乱码：--character-set-server=utf8
解决mysql中文乱码：--collation-server=utf8_bin
mysql表名不区分大小写：--lower_case_table_names=1
