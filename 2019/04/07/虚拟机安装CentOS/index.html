虚拟机安装CentOS

发表于 2019-04-07 | 更新于 2019-04-08 | 分类于 技术

1.设置VMware

编辑-->虚拟网络编辑器-->选择VMnet8-->修改模式为NET模式-->勾上将主机虚拟适配器连接到此网络，取消勾选使用本地DHCP服务-->点击NET设置-->记下子网ip，子网掩码，网关ip

2.修改CentOS的ip地址为静态ip地址

vim /etc/sysconfig/network-scripts/ifcfg-xxx

修改和添加以下内容：
BOOTPROTO=static
ONBOOT=yes
IPADDR=192.168.2.88  #以你自己的ip为准，必须在子网ip网段内
NETMASK=255.255.255.0  #以你自己的VMware的子网掩码为准
GATEWAY=192.168.2.2  #以你自己的VMware的网关ip为准
DNS1=114.114.114.114  #dns

重启网卡: systemctl restart network

3.修改ssh配置文件

安装ssh: yum install -y openssh-server

vim /etc/ssh/sshd_config

去掉以下代码前面的#：
Port 22
ListenAddress 0.0.0.0
ListenAddress ::
PermitRootLogin yes
PasswordAuthentication yes

重启ssh服务：systemctl restart sshd

4.更换基础源和其他设置

删除源并添加国内的源：
rm -rf /etc/yum.repos.d/*
vim /etc/yum.repos.d/CentOS-Base.repo
阿里云源：
# CentOS-Base.repo

[base]
name=CentOS-$releasever - Base
baseurl=https://mirrors.aliyun.com/centos/$releasever/os/$basearch/
gpgcheck=0

[updates]
name=CentOS-$releasever - Updates
baseurl=https://mirrors.aliyun.com/centos/$releasever/updates/$basearch/
gpgcheck=0

[extras]
name=CentOS-$releasever - Extras
baseurl=https://mirrors.aliyun.com/centos/$releasever/extras/$basearch/
gpgcheck=0

添加扩展源：
vim /etc/yum.repos.d/epel.repo
阿里云扩展源：
# epel.repo

[epel]
name=Extra Packages for Enterprise Linux $releasever - $basearch
baseurl=https://mirrors.aliyun.com/epel/$releasever/$basearch
gpgcheck=0

关闭SELinux：
setenforce 0
sed -i "s/SELINUX=enforcing/SELINUX=disabled/g" /etc/selinux/config

关闭防火墙：
systemctl stop firewalld.service
systemctl disable firewalld.service

5.安装java

vim /etc/profile

添加以下内容：
export JAVA_HOME=/usr/local/java/jdk1.8.0_171
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

更新环境变量：source /etc/profile
添加权限：chmod -R 777 /usr/local/java/jdk1.8.0_171

6.安装mysql

卸载系统自带的mariadb-lib：rpm -qa | grep mariadb

下载mysql：https://dev.mysql.com/downloads/mysql/
在下面的选择框做如下操作：
Select Operating System：Red Hat Enterprise Linux / Oracle Linux
Select OS Version：ALL
下载：
Download Packages：mysql-8.0.15-1.el7.x86_64.rpm-bundle.tar

下载mysql依赖：
http://mirror.centos.org/centos/7/os/x86_64/Packages/net-tools-2.0-0.24.20131004git.el7.x86_64.rpm
http://mirror.centos.org/centos/7/os/x86_64/Packages/libaio-0.3.109-13.el7.x86_64.rpm

安装：
rpm -ivh common.rpm
rpm -ivh libs.rpm
rpm -ivh client.rpm
rpm -ivh libaio.rpm
rpm -ivh net-tools.rpm
rpm -ivh server.rpm

初始化数据库：mysqld --initialize --user=mysql

查看初始化密码：cat /var/log/mysqld.log

启动mysql服务：systemctl start mysqld.service

修改密码：ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';

修改数据库表名不区分大小写：
vim /etc/my.cnf
添加：lower_case_table_names =1
重启mysql服务：systctl restart mysqld
查看修改结果(登录mysql后执行)：show variables like "%case%";

修改数据库编码：
vim /etc/my.cnf
添加：
[mysqld]
character-set-server=utf8
collation-server=utf8_general_ci
[client]
default-character-set=utf8
重启mysql服务：systctl restart mysqld
查看修改结果(登录mysql后执行)：status
