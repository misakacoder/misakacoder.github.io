常用的高级搜索方法

发表于 2019-03-29 | 分类于 技术

1.双引号

将需要搜索的文本用双引号引起来，搜索的文本不会被搜索引擎拆分

2.加号

java +spring
搜索关键字为java，并且返回的搜索结果包含spring

3.减号

java -spring
搜索关键字为java，并且返回的搜索结果不包含spring

4.星号

java*资料
搜索结果可能是java复习资料，java学习资料

5.intitle

java intitle:黑马程序员
搜索关键字为java，并且返回的搜索结果的标题包含黑马程序员

6.inurl

java inurl:www.zhihu.com
搜索关键字为java，并且返回的搜索结果的网址包含www.zhihu.com

7.filetype

filetype:doc java
搜索关键字为java，并且返回的搜索结果是包含java的doc文档

8.site

java site: www.zhihu.com
搜索关键字为java，限定搜索范围为仅在www.zhihu.com这个网站里面搜索
